# Deleting a password

1.  Login and/or go to the password workspace
2.  In the list, click on the password(s) you wish to delete
3.  Click on the “More” button on top of the password list
4.  Select the “Delete” option.
5.  Click “Delete” to confirm.

## Pro tip:

- Alternatively you can right click on a password and then choose the ”Delete” option in the contextual menu.
- You can also delete a whole folder of passwords by:
    - clicking on the ”...” button next to the folder name and select the ”Delete” option
        - or right click on any folder and choose the ”Delete” option in the contextual menu
    - checking ”Also delete items inside this folder”, otherwise all passwords will be moved in the root folder
    - clicking ”Delete” to confirm
