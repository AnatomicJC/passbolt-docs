# Editing a password

<div style={{display: "flex", gap: "1rem"}}>
  <Chips link="https://www.passbolt.com/ce/docker" isUnderMainTitle={true}>CE</Chips>
  <Chips link="https://www.passbolt.com/pricing/pro" isUnderMainTitle={true}>Pro</Chips>
  <Chips link="https://www.passbolt.com/pricing/cloud" isUnderMainTitle={true}>Cloud</Chips>
</div>

1.  Login and/or go to password workspace
2.  Select the password from your list
3.  Click the “Edit” button on top of your password list
    - You can also right click on the password and select “Edit“ from contextual menu
4.  Edit your password and press the save button
5.  Wait till Encryption is done

## Pro Tip:

- Press on the “Eye” button to check the edits made to your password
